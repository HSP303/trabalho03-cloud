Sobre o projeto

Ambiente DevOps para videomonitoramento

Descrição

Este site representa a camada de publicação de uma solução de videomonitoramento. O ambiente é executado em container Ubuntu, utilizando Apache para servir os arquivos estáticos e Shell Script para automatizar tarefas de administração Linux.

Relação com Cloud Computing

Em um cenário real, câmeras poderiam enviar eventos, alertas e gravações para uma infraestrutura cloud. Neste trabalho, a estrutura foi simulada com diretórios de câmeras, gravações, alertas, logs, backups e publicação, demonstrando práticas básicas de operação.
