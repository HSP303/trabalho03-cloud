CloudVision Início Sobre

Sistema para Videomonitoramento em Cloud

Painel operacional de câmeras, alertas e gravações

Ambiente estático publicado via Apache para simular a interface pública de uma solução de videomonitoramento baseada em infraestrutura Linux containerizada.

CAM01 - Entrada

Status: online. Monitoramento contínuo da entrada principal.

CAM02 - Recepção

Status: online. Área administrativa monitorada em tempo real.

CAM03 - Estoque

Status: manutenção. Necessita validação técnica no ambiente.

Rotinas automatizadas

  • Atualização do sistema Linux via Shell Script.
  • Instalação e validação do Apache com pacote complementar ffmpeg.
  • Backup das configurações, dados, alertas e gravações simuladas.
  • Monitoramento de CPU, memória, disco e status do Apache.
  • Controle de usuários, grupos e permissões seguras.
